plotter-sketches

To run it locally:
- install nodejs / npm
- install globally canvas-sketch (npm install canvas-sketch -g)
- run canvas-sketch: canvas-sketch sketch-voronoi-manhattan.js --open --output=export/ (command is also duplicated in comments)
- enjoy
